## 编辑器
### Vim
#### 命令模式
启动时
- i 切换到输入模式
- x 删除字符
- : 切换到底线命令模式

#### 输入模式
- Insert 切换输入/替换模式
- Esc 切换到命令模式

#### 底线命令模式
前面加冒号`:`
- q 退出
- q! 强制退出
- e! 放弃修改，重新开始编辑
- w [FILE] 保存文件
- w! 强制保存
- x 或 wq 保存并退出
- wq! 强制保存并退出
